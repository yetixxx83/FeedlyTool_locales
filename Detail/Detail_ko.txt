현재 다음과 같은 기능이 있습니다.
· Fullfeed (iframe 원래 페이지를 엽니 다.)
· Readability · Instapaper · DiffBot · FeedlyTool Server (각각의 서비스에서 콘텐츠를 가져옵니다.)
· AddThis 버튼 (www.addthis.com의 공유 버튼을 선택합니다.)
· 멋진 스타일 적용 (아래 참고로 드린 스타일을 기재하고 있습니다.)
· "여기까지 읽은"(나열시 제목 녹색 아이콘을 클릭하여 읽은 상태로 표시 할 수 있습니다.)
· 파비콘 표시 (목록 표시시 앞에 파비콘을 표시합니다.)
· 글쎄 북마크 수 표시
· 광고 피드 삭제 (토큰을 취득한 경우 백그라운드에서 수행됩니다.)
· 읽지 않은 수 아이콘 표시 (토큰의 취득이 필요합니다.)
아이콘 클릭으로 Feedly 열기
· RSS 전달 된 페이지에서 Feedly의 등록 링크 표시 (아이콘 클릭으로 표시됩니다.)
· Index 페이지 (cloud.feedly.com / # index)에 업데이트 일 이후의 날짜를 표시 (토큰의 취득이 필요합니다.)
옵션 페이지에서 토큰 및 설정 삭제
· "Save for Later"에서 "여기까지 해제"
· Fullfeed · Readability · Instapaper · DiffBot · FeedlyTool을 각각 1 ~ 5의 단축키에 할당했습니다.
・Google 사이트 검색 상자를 사용하여 피드 검색

주의
Adblock Plus를 사용하는 경우 Fullfeed가 열립니다 きる 전에 닫아 버리는 페이지가 있습니다. 공식 기능의 preview에서도 같은 현상이 일어나므로, 걱정마하는 것으로 부탁합니다.
Chrome에서 "https 페이지에서 iframe으로 http 페이지를 표시하는"것이 금지 된 것입니다. https 연결을 사용하시는 분은 Fullfeed가 기존에 나타나지 않습니다. 양해 바랍니다.
